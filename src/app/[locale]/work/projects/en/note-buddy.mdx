## Overview

Who hasn't wanted to take notes but didn't want to spend time writing? Taking notes, drafting essays, and writing down ideas should be quick and intuitive.
[Note-Buddy](https://note-buddy.app) is a web application that allows you to take notes and draft documents quickly and efficiently.

## Who is it for?

This tool is designed for students, engineers, doctors, etc.

There are many use cases:

- Taking lecture notes
- Writing an internship report
- Writing a thesis
- Writing an email
- ...

## The problem it solves

Creating notes quickly and efficiently.

## How does it work?

![Note-Buddy prez](/images/projects/notebuddy/prez-ask-ai.gif)

Ask the AI to write your note, article, report, etc. Then you can ask it to improve, rephrase, explain, correct, or find synonyms for words.
You can also ask it to translate into different languages or change the text style.

To sign up, simply connect via [the website](https://note-buddy.app/signup).

## Main Features

### Modify Content

- **Improve content:** Enhance content by adding details, correcting spelling mistakes, finding synonyms, etc.
- **Correct content:** Fix spelling, grammar, and syntax errors, etc.
- **Simplify content:** Simplify content by removing unnecessary words, summarizing, clarifying sentences, etc.
- **Add details:** Add details to explain the content.

### Generate Content

- **Auto-completion:** The AI can complete your sentence based on what you've written.
- **Summarize:** Summarize the content.
- **Translate:** Translate content into different languages.
- **Change style:** Change the content style, using more formal or technical vocabulary.
- **Explain:** Explain the content.

## Why use Note-Buddy?

- It offers features that no one else offers.
- It's free but limited.

## Technologies Used

- **[Next.js](https://nextjs.org)**: Framework used for application development.
- **[OpenAI](https://openai.com)**: AI used for content generation.
- **[Vercel](https://vercel.com)**: Platform used for application deployment.
- **[TailwindCSS](https://tailwindcss.com)**: Framework used for application design.
- **[Shadcn/UI](https://ui.shadcn.com)**: Component library used for application design.
- **[Zod](https://zod.dev)**: Library used for data validation.
- **[Supabase](https://supabase.com)**: Database service used for data storage.

## Challenges and Learnings

Developing Note-Buddy is a personal project and takes a lot of time. I learned how to design a web application with AI-generated content.

## Results

Note-Buddy is still in beta and is used by several people in my circle of friends. But for now, it isn't used by thousands of people.
