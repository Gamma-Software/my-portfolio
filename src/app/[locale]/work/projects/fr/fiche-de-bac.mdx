## Aperçus

Ce projet est LA toute première application Android que j'ai créée. J'ai découvert Eclipse (à l'époque Android Studio n'existait pas) et j'ai découvert le développement Android.
Le but était de créer une application qui recensait toutes mes fiches de bac de physique-chimie (J'ai eu 17/20 à l'épreuve de bac de physique-chimie).

## Fonctionnalitées principales

- Fiches de bac
- Fiches de cours
- Fiches de TP
- Fiches de résumé
- Test si tu avais le bac ou pas

## Technologies Utilisées

- Eclipse
- Android
- Java pour le développement

## Challenges et apprentissages

- Découvrir le développement en général
- Découvrir le développement Android
- Découvrir Eclipse
- Découvrir SQLite
